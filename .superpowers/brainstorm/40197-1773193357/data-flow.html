Error Handling

Sync failure
UI shows "Last synced X ago" badge. Goes amber after 1 day, red after 2. Doesn't break the page — shows last good data with a staleness timestamp. Slack alert fires via existing n8n hook.
Bad URL params
Invalid creator ID or malformed dates → fall back to defaults (30D, first creator). Never throws a 500 to the UI.
No data for range
Platform card shows empty state ("No data for this period") — not an error, not a spinner. Clean empty state per platform.
LTK token expired
LTK API returns 401 → catch in sync job, fire Slack alert with instructions to re-auth. Token rotation runs every 3h via Railway scheduler to prevent this.
Missing post thumbnail
Content card falls back to a gradient placeholder (by content type) — same as the mockup. Never a broken image.
Attribution match fails
Post shows "No link" state — still visible in the grid with engagement metrics. Not hidden, not errored. Revenue shown as "—".
