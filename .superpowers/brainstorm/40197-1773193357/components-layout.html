Components built or modified

NEW — useFilters() hook
Reads URL params, exposes creator + date range, provides setCreator() and setDateRange(). All components import this — nothing else needed.
MODIFIED — app/dashboard/layout.tsx
Add the sticky filter bar with CreatorSelector + DateRangePicker. Renders above every dashboard page automatically.
NEW — DateRangePicker component
Preset chips + Custom calendar. On change calls setDateRange(). Highlights active preset. Shows resolved date label (Feb 10 – Mar 12).

Add creatorId + startDate + endDate query params. DB queries updated with correct WHERE clauses per platform. LTK route adds pagination loop.
